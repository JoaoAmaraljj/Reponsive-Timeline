My Fullstack journey

  • May HTML Language

    Imagine HTML as the skeleton or blueprint of a webpage. It defines the content structure and layout of a webpage using tags.

  • June CSS Language

    CSS is like the fashion stylist for your webpage built with HTML. It controls the visual appearance of the webpage, including fonts, colors, layout, and positioning of elements.

  • July JavaScript Programming

    JavaScript adds interactivity and dynamism to webpages. It's like the programmer that brings the webpage to life. Imagine click events, animations, or forms that validate user input - these are all powered by JavaScript.

  • AugustDatabase-SQL

    SQL databases are like digital filing cabinets for organized data, storing information in tables with rows and columns. SQL is a special language used to access and manage this data, allowing users to create, read, update, and delete records efficiently.

  • September C# Programming

    C# is a powerful, beginner-friendly language for building many kinds of software. Think web apps, desktop programs, games, and business applications - all written in C#.

  • October Python Programming

    Programming language known for its readability and beginner-friendliness, used for many things. It can build websites, crunch data, automate tasks, and even power AI!

  • November React JS

    React.js is a JavaScript library for building user interfaces (UIs), It allows developers to create reusable components that update efficiently, making it ideal for building dynamic and interactive web applications.

  • December Node JS

    Node.js is used for building server-side applications. It's a popular choice for applications that need to handle many concurrent connections or real-time data.
